**scripts/register.ps1** -- Full dev registration workflow (per D-07):

```powershell
# Parameters:
param(
    [string]$BuildDir = "$PSScriptRoot\..\build",
    [string]$PackageDir = "$PSScriptRoot\..\package",
    [string]$CertSubject = "CN=ClaudeFromHere",
    [string]$CertFriendlyName = "ClaudeFromHere Dev Cert",
    [switch]$SkipBuild
)
```

**Step 0 -- Build DLL (unless -SkipBuild):**
- Resolve full path for BuildDir and PackageDir using `Resolve-Path` or `[System.IO.Path]::GetFullPath()`
- Create build directory if it doesn't exist
- Run CMake configure: `cmake -S "$PSScriptRoot\.." -B "$BuildDir" -G "Visual Studio 17 2022" -A x64`
- Run CMake build: `cmake --build "$BuildDir" --config Release`
- Verify `$BuildDir\Release\ClaudeFromHere.dll` exists (CMake with VS generator puts output in Release/ subfolder)
- Copy the DLL from `$BuildDir\Release\ClaudeFromHere.dll` to `$BuildDir\ClaudeFromHere.dll` so the MSIX external location works

**Step 1 -- Certificate generation (per D-08):**
- Check `Cert:\LocalMachine\TrustedPeople` for existing cert with Subject matching `$CertSubject`
- If cert exists, write "Certificate already imported, skipping..." and retrieve its thumbprint
- If cert does NOT exist:
  - Write "Generating self-signed certificate..."
  - `$cert = New-SelfSignedCertificate -Type Custom -Subject $CertSubject -FriendlyName $CertFriendlyName -KeyUsage DigitalSignature -CertStoreLocation "Cert:\CurrentUser\My" -TextExtension @("2.5.29.37={text}1.3.6.1.5.5.7.3.3")`
  - Generate a random password: `$password = [System.Guid]::NewGuid().ToString()`
  - Export PFX: `Export-PfxCertificate -Cert $cert -FilePath "$BuildDir\ClaudeFromHere.pfx" -Password (ConvertTo-SecureString -String $password -Force -AsPlainText)`
  - Save password to file: `$password | Out-File "$BuildDir\cert-password.txt"`
  - Check if running elevated. If NOT, write "Admin elevation required for cert import to LocalMachine\TrustedPeople..." and use `Start-Process powershell -Verb RunAs -ArgumentList "-Command Import-PfxCertificate -FilePath '$BuildDir\ClaudeFromHere.pfx' -CertStoreLocation 'Cert:\LocalMachine\TrustedPeople' -Password (ConvertTo-SecureString -String '$password' -Force -AsPlainText)"` then `-Wait`
  - If already elevated, import directly

**Step 2 -- Copy assets to build directory:**
- Copy `assets\claude.ico` to `$BuildDir\claude.ico` (so GetIcon can find it relative to DLL)
- The MSIX package needs the manifest. Pack from `$PackageDir` which contains AppxManifest.xml and Assets\StoreLogo.png

**Step 3 -- Pack MSIX:**
- `$MakeAppx = "C:\Program Files (x86)\Windows Kits\10\bin\10.0.26100.0\x64\makeappx.exe"`
- `& $MakeAppx pack /o /d $PackageDir /nv /p "$BuildDir\ClaudeFromHere.msix"`
- Check `$LASTEXITCODE -eq 0`, exit with error message if not

**Step 4 -- Sign MSIX:**
- `$SignTool = "C:\Program Files (x86)\Windows Kits\10\bin\10.0.26100.0\x64\signtool.exe"`
- Read password from `$BuildDir\cert-password.txt` if PFX exists there
- If PFX doesn't exist in BuildDir (cert was already imported), find it: look for the cert thumbprint in `Cert:\LocalMachine\TrustedPeople`, export to PFX from `Cert:\CurrentUser\My` using that thumbprint
- `& $SignTool sign /fd SHA256 /a /f "$BuildDir\ClaudeFromHere.pfx" /p $password "$BuildDir\ClaudeFromHere.msix"`
- Check `$LASTEXITCODE -eq 0`

**Step 5 -- Unregister existing (idempotent):**
- `$existing = Get-AppxPackage -Name "ClaudeFromHere" -ErrorAction SilentlyContinue`
- If `$existing`: `Remove-AppxPackage $existing.PackageFullName`

**Step 6 -- Register with ExternalLocation:**
- `Add-AppxPackage -Path "$BuildDir\ClaudeFromHere.msix" -ExternalLocation "$BuildDir"`
- Check for errors

**Step 7 -- Restart Explorer:**
- `Write-Host "Restarting Explorer to load extension..."`
- `Stop-Process -Name explorer -Force; Start-Sleep -Seconds 2`
- Explorer auto-restarts on Windows 11

**Step 8 -- Success message:**
- `Write-Host "Registration complete! Right-click a folder to test."`

**scripts/unregister.ps1** -- Clean teardown (per D-07):

```powershell
param(
    [switch]$RemoveCert,
    [string]$CertSubject = "CN=ClaudeFromHere"
)
```

- Remove AppxPackage: `Get-AppxPackage -Name "ClaudeFromHere" | Remove-AppxPackage`
- If `-RemoveCert`: remove from `Cert:\LocalMachine\TrustedPeople` (requires elevation) and `Cert:\CurrentUser\My`
- Restart Explorer: `Stop-Process -Name explorer -Force`
- Write "Unregistered. Context menu item removed."

**Error handling throughout both scripts:**
- Check `$LASTEXITCODE` after every external tool call
- Use `$ErrorActionPreference = "Stop"` at the top
- Wrap tool invocations in try/catch where appropriate
- Print clear error messages with the failing step name
